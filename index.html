Showcase video

As part of my university web foundations assignment, a clear criterion was set to what should be included in the website, one the of the elements was a short self-made video that when clicked functions correctly allowing the user to watch the footage pause and change the volume, this was a key part of the criteria so I decided to make sure it was in plain view on the first page of the website. Feel free to check it out.
